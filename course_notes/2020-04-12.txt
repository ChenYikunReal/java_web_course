1 动态网站和静态网站对比
静态网站 内容是固定的，不能实现交互操作。
动态网站 内容不是固定的，能够实现交互操作。  --应用这种

2、bs与cs结构程序的对比
bs brower server 浏览器 服务器    ---学习这个
 程序完全部署在应用程序服务器上
 用户通过浏览器就能访问服务器端
 服务器端更新客户端不需要更新也能访问
 页面一般不会像cs程序那么绚丽
cs client server 客户端 服务器
 用户必须安装客户端才能够访问服务端
 服务器端更新客户端必须更新才能访问
 页面一般做的比较绚丽
 如：电脑版的qq、大型游戏

学习使用servlet、jsp等技术实现bs结构程序的开发
 使用 html css JavaScript jquery 数据库 servlet jsp 技术

3、uri与url
uri 统一资源标识符
 里面包含url
url 统一资源定位符（路径）
 协议://服务器ip地址或主机名：端口号/项目名/路径
 http://192.168.1.100:8080/test/index.jsp
 ftp://localhost:8088/test
 注意：如果服务器是本机 ip地址可以写 localhost或127.0.0.1或本机ip地址
 http://www.163.com

4 http协议 超文本传输协议
  位于应用层的一个协议
  请求与响应都是基于http协议的
  可靠的
  默认端口80
  不记录状态
5 http请求
  由请求行、请求消息头、空行、请求体所组成
  请求行由请求方法、uri、协议/版本组成
6 http响应
  由状态行、响应头、响应正文所组成
  状态行由 协议/版本、状态码、响应字符串。
  状态码--200 成功、404路径错误、500服务器异常

9 修改tomcat的端口号
默认端口号是8080
修改--在tomcat目录--conf--server.xml修改端口号
  <Connector port="8080" protocol="HTTP/1.1"
               connectionTimeout="20000"
               redirectPort="8443" />
